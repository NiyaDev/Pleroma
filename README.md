
### Pleroma
A group of libraries used for game development

#### Libraries
##### [Bulk](bulk.c3l/README.md)
A custom made resource packer.
##### [Keybind](keybind.c3l/README.md)
A keybinding system that implements Raylib.
##### [Raylib](raylib.c3l/README.md)
A custom implementation of Raylib using wrappers around bindings.
